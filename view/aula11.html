O que seria um web designer?

Um web designer é um profissional responsável pela criação e desenvolvimento do layout,
design visual e estrutura de um sítio Web. O trabalho de um web designer envolve a combinação de
habilidades de design gráfico e conhecimento técnico para criar sítios Web que sejam esteticamente agradáveis,
funcionais e fáceis de usar.

Qual o papel do web designer?

  • Criador - concepção e o desenvolvimento de páginas Web;
  • Auditor - colabora no levantamento de requisitos no desenvolvimento do software Web;
  • Consultor - participa na avaliação e fiscalização da qualidade do sítio Web;

Mais especificamente, a função de um Web Designer passa por definir a estrutura (arrumação dos
conteúdos e a navegabilidade), o aspecto visual e a usabilidade de um sítio Web, ou seja, produzir um
modelo atrativo para o utilizador, que tenha lógica de navegação e que tenha a informação visível realmente necessária.

o Web Designer deve seguir algumas etapas das quais se destacam:

  • levantamento dos requisitos;
  • Desenho do modelo
  • Implementação

O que é tipografia?

Tipografia é conjunto de procedimentos artísticos e técnicos que abrangem as diversas etapas da
produção gráfica (desde a criação dos caracteres até a impressão e acabamento)
